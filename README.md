# Turbo Boost Arduino

This is a repository with Arduino code and KiCad schematics for a turbo boost sensor using:
- MPX4250AP (pressure sensor)
- SSD1306 (display)
- LM2596 (step-down converter)
- NodeMCU (microcontroller)
